# 요구사항
Todo list 
- 사용자는 회원가입을 할 수 있다.
- 사용자는 회원탈퇴를 할 수 있다.
- 사용자는 로그인을 할 수 있다.
- 사용자는 Todo 항목을 등록 할 수 있다.
- 사용자는 Todo 항목을 상태를 수정 할 수 있다.


# 개발 환경 
- **Java Version**: 17
- **Spring Boot Version**: 3.3
- **Database**: MySQL 8.0
- **ORM**: JPA
